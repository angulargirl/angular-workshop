angular-workshop
================

AngularJS training source code

Get started:

1. Make sure you have Node/NPM installed. (http://nodejs.org/)

2. Clone this project

3. cd to API and do npm install

```
cd api
npm install
```

4. cd to app and install dependencies
```
cd app
npm install
bower install
```

4. run api
```
cd api
node server
```

5. run app
```
cd app
gulp
```
